== Description ==

This is a widget that allows you to search the <a href="http://www.rhymebox.com">rhyming dictionary</a> <a href="http://www.rhymebox.com">rhymebox.com</a>. You can choose which language your users will search rhymes in, at the moment english and german are available.

== Installation ==
Unzip, upload to your plugin directory, add the widget to your site.
<ol>
<li>Upload rhymebox-widget to the `/wp-content/plugins/` directory</li>
<li>Activate the plugin through the 'Plugins' menu in WordPress</li>
<li>Activate the widget through the 'Design' -> 'Widgets' menu in WordPress</li>
<li>Enjoy!</li>
</ol>
